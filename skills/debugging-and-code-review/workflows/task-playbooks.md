# Task Playbooks

## Bug Fix

- Reproduce
- Capture logs
- Isolate cause
- Fix minimally
- Add regression test

## Regression Investigation

- Compare recent changes
- Confirm behavior drift
- Identify introduced assumption or dependency
- Validate fix against old and new paths

## Code Review

- Verify correctness
- Review limits, security, and maintainability
- Confirm tests are meaningful
- Approve only with confidence
