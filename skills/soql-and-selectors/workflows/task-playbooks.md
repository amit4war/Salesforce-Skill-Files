# Task Playbooks

## Playbook A: Add New Selector

1. Define method contract.
2. Implement selective SOQL.
3. Add tests.

## Playbook B: Refactor Inline SOQL

1. Identify duplicate query blocks.
2. Consolidate into selector.
3. Update callers and regression tests.

## Playbook C: Security Hardening

1. Review sharing mode and field access.
2. Apply security enforcement.
3. Validate behavior under restricted user context.
